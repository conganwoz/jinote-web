## Jinote web

This is FE of the note taking app. it allows to store data on client device and transfer data throught a P2P connection (WebRTC)

It is completely sercue and private :)

## Technologies
- React 17.0.2
- Ant design

## Setup
```npm install```

## Launch
```npm run start```\
--> open url: http://localhost:3000
